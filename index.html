Frequently Asked Questions

What image formats are supported?

AniResizer supports JPG, JPEG, PNG, and WebP formats. These are rendered directly in your browser without any server upload.

Does it compress images too?

Yes! When you enable "Reduce File Size", we compress the image using optimized encoding. This reduces file size significantly while maintaining visual clarity.

Is this tool safe for private images?

Absolutely. All processing happens locally in your browser. No files are uploaded to any server, making it completely private and secure.

Can I use it offline?

Yes! After your first visit, the app works offline. You can even download the source code and run it from your local machine without any internet connection.
